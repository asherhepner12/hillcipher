# Hill Cipher Script

This is a Java script that implements the Hill Cipher in order to encrypt and decrypt a string.

A key string is used that must be:
 - Of sqrt(n) size (eg. 9 char string for a 3x3 matrix, 4 char string for 2x2 matrix, etc.)
 - The matrix's determinant must have an inverse mod 26

I have packaged the script into a standalone .exe for my less tech-savvy friends that includes the JRE, just download and go!
Or if you like, just run the script yourself.
